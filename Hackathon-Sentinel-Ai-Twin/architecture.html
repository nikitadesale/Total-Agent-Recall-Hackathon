🔄 How a Request Flows
1
User asks a question
Chat input in the React UI — e.g. "Should I order DoorDash tonight?"
2
Axios → POST /chat
Full message history sent to FastAPI with include_vault: true
3
Smart context routing
Query intent parsed → only relevant domains loaded (saves tokens)
4
HydraDB recall
Past decisions and patterns retrieved to give the model memory
5
Kimi K2.5 reasons
GMI Cloud model receives system prompt + vault context + history
6
Response + memory stored
Reply returned to UI · Turn saved to HydraDB · Vault refreshes
